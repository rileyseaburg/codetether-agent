## Bug 2: Path traversal via session ID
**File:** src/session/persistence.rs — the `session_path` function
**Fix:** Validate the session ID is alphanumeric/dash/underscore only:
```rust
pub(crate) fn session_path(id: &str) -> Result<PathBuf> {
    // Reject path traversal attempts
```

## Bug 3: Timing leak in constant-time compare
**File:** Find src/server/auth/ — look for constant_time_eq or compare function
**Fix:** Fold length into diff instead of early return:
```rust
pub fn constant_time_eq(a: &[u8], b: &[u8]) -> bool {
    let len_diff = (a.len() ^ b.len()) as u8;
    let mut diff = len_diff;
    for i in 0..a.len().max(b.len()) {
        let left = a.get(i).copied().unwrap_or(0);
        let right = b.get(i).copied().unwrap_or(0);
        diff |= left ^ right;
    }
    diff == 0
}
```

## Bug 4: Plugin hash compared with String ==
**File:** src/plugin_marketplace/verify.rs — verify_hash function
**Fix:** Use constant-time comparison for the hash:
```rust
pub fn verify_hash(data: &[u8], expected_hex: &str) -> bool {
    use sha2::{Sha256, Digest};
    let mut hasher = Sha256::new();
    hasher.update(data);
    let result = hasher.finalize();
    let Ok(expected) = hex::decode(expected_hex) else { return false };
    // Constant-time comparison
    let mut diff = 0u8;
    for (a, b) in result.as_slice().iter().zip(expected.iter()) { diff |= a ^ b; }
    diff == 0 && result.as_slice().len() == expected.len()
}
```
